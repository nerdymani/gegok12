Class Detail

Livewire component responsible for displaying add-on details and handling the purchase flow in the Admin panel.

This component:

  • Fetches add-on details
  • Loads available payment gateways
  • Handles add-on purchase requests
  • Redirects to payment gateways (Razorpay / Bank)
